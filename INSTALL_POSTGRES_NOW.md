# Install PostgreSQL - Step by Step

## Step 1: Open PowerShell
- Press `Win + X`
- Click "Windows PowerShell" or "Terminal"
- Make sure you're in your user folder (like `C:\Users\HP>`)

## Step 2: Install PostgreSQL
Run this EXACT command:

```powershell
winget install PostgreSQL.PostgreSQL.15
```

**Where to run:** From ANY folder (PowerShell home folder is fine)

**What will happen:**
- It will download PostgreSQL (~200MB)
- It will ask you to accept the license - type `Y` and press Enter
- It will install PostgreSQL
- **IMPORTANT:** During installation, a window will pop up asking for a password
- **Set password to:** `postgres` (remember this!)

## Step 3: Wait for Installation
- Wait until you see "Successfully installed" message
- This takes 2-5 minutes

## Step 4: After Installation
Once installation is complete, come back here and I'll help you set up the database!
